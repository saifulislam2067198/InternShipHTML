HTML Interview Questions Answers

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML.

CSS allows you to control the layout, colors, fonts, and overall appearance of a web page. It works by associating rules with HTML elements, which specify how those elements should be displayed on the screen.

What is HTML?

HTML (Hypertext Markup Language) is the standard markup language used to create web pages. It provides the structure and content of a web page, allowing you to define elements such as headings, paragraphs, images, links, and more.

What is the DOM?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of an HTML document as a tree-like structure, where each element is a node in the tree. The DOM allows developers to manipulate the content and structure of a web page dynamically using JavaScript.

What is the difference between HTML and XHTML?

HTML (Hypertext Markup Language) is the standard markup language for creating web pages, while XHTML (eXtensible Hypertext Markup Language) is a stricter and more XML-based version of HTML. XHTML requires proper document structure, closed tags, and lowercase element names, making it more consistent and easier to parse.
